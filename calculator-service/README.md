# Go gRPC

## References

- [gRPC](https://grpc.io/)

## Generate proto

```sh
$ sh generate.sh
```

## Greet

Run Greet server

```sh
$ go run greet/greet_server/server.go
```

Run Greet client

```sh
$ go run greet/greet_client/client.go
```

## Calculator

Run Calculator server

$ go run calculator/calculator_server/server.go
